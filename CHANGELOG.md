# Changelog — Книготрек

Формат: [Keep a Changelog](https://keepachangelog.com/ru/), версии: [SemVer](https://semver.org/lang/ru/).

## [1.0.0] — готовится к релизу

Первый публичный релиз. Книготрек — бесплатный open-source (MIT) трекер прогресса для писателей. Все функции открыты, без ограничений и подписок.

### Возможности
- 📈 **Трекинг прогресса** — ежедневные записи символов, статистика, календарь-теплокарта, рекорды и серии дней
- 📚 **Проекты** — цели по объёму, дедлайны с адаптивным переносом, фазы «черновик/редактура», главы со статусами
- ⏱️ **Режим фокуса** — таймер письма, сессии с тегами и заметками, аналитика скорости
- 🤖 **Ассистент** — прогноз финиша, дневной план, аналитика тегов и персональные рекомендации, цели-челленджи
- 📄 **Отчёты** — PDF-отчёты (день/неделя/месяц/проект/год), share-картинки, «паспорт писателя», итоги года
- 📝 **Заметки** — быстрые заметки и идеи
- 🌐 **Два языка** — русский и английский
- 🌙 **Тёмная тема**
- 💾 **Локальное хранение** — SQLite на десктопе (Electron), автобэкапы; работает и в браузере (localStorage)
- ☁️ **Опциональная синхронизация** — через собственный проект Supabase (по желанию, через `.env`)

### Технологии
- Electron + React + TypeScript + Vite, better-sqlite3, Vitest, Tailwind CSS
